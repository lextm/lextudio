\chapter{Installation/Uninstallation}
\emph{''Be TIRED of hard-to-install plug-ins?''}

CBC is easy to install/uninstall.
\piccbc
\newpage

\section{Installation Guide}
\begin{flushleft}
\begin{enumerate}
\item Shut down IDE before installation.
\item Unzip the CBC zip archive, and run setup.exe. Follow the instructions
of the installer.
\item Next time you open IDE, a menu item named ''LeXtudio'' is added to your
IDE main menu.
\end{enumerate}
\end{flushleft}

\section{Uninstallation Guide}
\begin{flushleft}
\begin{enumerate}
\item Shut down IDE before uninstallation.
\item Open Control Panel of your Windows OS. In the dialog, choose Code Beautifier Collection icon and click Uninstall. Follow the instructions.
\item Next time you open IDE, the menu item named ''LeXtudio'' is removed from
your IDE main menu.
\end{enumerate}
\end{flushleft}
