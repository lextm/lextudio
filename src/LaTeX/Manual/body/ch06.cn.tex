\chapter{致无法使用\ CBC 6\ 的朋友们}
\emph{“我本该在你的身旁\dots”}

依然祝你快乐。
\piccbc
\newpage
如果你不是\ CodeGear RAD Studio\ 的用户，很抱歉你不能享受到\ CBC 6\
可能带给你的种种欢乐。但是，这里仍然有一些对你有用的信息，使得你可以享受到自动化美化代码工具的使用快感。下面列出的工具都是免费的，其中一些还开放源码。
\begin{itemize}
  \item 如果~Borland Developer Studio (BDS 1.0 - 4.0)~还是你正在使用的~IDE，你可以下载~CBC
  5.3.3，地址在旧的主页，
  \hyperlink{http://gforge.oss.org.cn/projects/codebeautifiers}
\item 如果你在找寻\ Borland Delphi 5/6/7 可以使用的插件，你可以在\ JEDI Code Format\ 的站点上找到，网址是
\hyperlink{
http://jedicodeformat.sourceforge.net/
}
\item Borland C++Builder 5/6\ 的用户可以试试\ C++ Formatting Expert。我没有用过它，也不知道它是不是支持更老版本的\ BCB。

主页:
\hyperlink{
http://www.frasersoft.net/program/
}
ZIP包:
\hyperlink{
http://www.frasersoft.net/program/fsexpert.zip
}

\end{itemize}

当然除此以外还有十分丰富的格式化工具，你可以在\underline{Google.com}上面找到。
