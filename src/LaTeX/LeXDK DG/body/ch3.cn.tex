
\chapter{用语}
\newpage

\begin{description}
\item [代码库]
这里指的是\ Sharp Builder Tools 3.1.0.0。

\item [Hervieux DNA]
也是指的\ Sharp Builder Tools 3.1.0.0。

\item [LeXDK]
包含在\ Lextm.LeXDK.Core.dll\ 中的代码。

由一系列的类组成。

Code Beautifier Collection (5\ 之后的版本)都是利用这个\ SDK\ 开发的。
\item [Minus]
扩展\ SDK\ 库的代码库可以被称作\ Minus。
\item [Plus]
利用\ LeXDK\ 开发的代码库。

它们被称作CBC/LeXDK\ 的''Plus''.
\item [特性]
plus\ 中的一些特殊类。

它们都是\ CustomFeature\ 的继承类。

它们被称作''特性(Features)''。

\end{description}
